Is ArchiCAD connected to Claude?

A 15-second check — no technical knowledge needed.

If Claude says it can only use a “Linux container” or can’t see your computer:

That just means the ArchiCAD connection didn’t load. This guide runs an automatic check and saves a small report to your Desktop. Email that report back and we’ll have it fixed quickly.

First, a 10-second sanity check

Make sure you are using the installed Claude Desktop app — a program in its own window — not the Claude website in a web browser (Chrome/Edge). The website cannot reach your computer and will always give the “Linux container” message.

Don’t have the app? Install it free from https://claude.ai/download, then re-run the installer.

Run the automatic check

1
Open PowerShell. Click the Windows Start button, type powershell, and press Enter. A dark blue window opens. (This is the same window you used to install.)
2
Copy the line below (triple-click it to select, then Ctrl‑C), click into the blue window, paste with Ctrl‑V, and press Enter.
▸ the one-line check — copy all of itirm https://raw.githubusercontent.com/agmurf/tapir-archicad-MCP/master/diagnose.ps1 | iex
3
Wait about 15 seconds. You’ll see a list of checks with [ OK ] or [PROBLEM] next to each, and a plain-English summary at the bottom.
4
Email the report. A file named Architech-Diagnostic-Report.txt is now on your Desktop. Attach it to a reply — that tells us exactly what to fix.

Prefer one click? Use the tool file instead

If copying and pasting feels fiddly, download this small file and double-click it — it runs the exact same check:

⬇ Download Diagnose-ArchitechMCP.bat

Windows may show a blue “Windows protected your PC” box because the file is new — click More info → Run anyway. It only reads your settings and writes the report; it changes nothing.

The two most common fixes

While you wait to hear back, these solve it the vast majority of the time:

  • Fully restart Claude Desktop. Closing the window isn’t enough — right-click the Claude icon in the system tray (bottom-right of the screen, near the clock) and choose Quit, then open it again. The ArchiCAD tools only appear after a full restart.
  • Re-run the installer. If the first install was interrupted, the connection never finishes setting up. Running the installer again is safe and fixes a half-finished install.

After either fix, open Claude Desktop and ask: “What MCP tools do you have?” If it lists ArchiCAD tools, you’re connected — open a project in ArchiCAD and you’re ready to go.
